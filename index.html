🕹️ Anomaly.exe

인간이 AI인 척 연기하여 시스템을 통과하는 5단계 미션 기반의 웹 게임 구현 (리버스 튜링 테스트)

React Node.js Firebase Gemini API
🐙 GitHub ▶ 시연영상

프로젝트 개요

AI가 인간처럼, 인간이 AI처럼 행동해야 하는 리버스 튜링 테스트 컨셉의 웹 게임입니다. 플레이어는 감정을 배제한 논리적인 답변으로 시스템의 AI 유사도 판별을 통과해야 하며, 총 5단계의 미션을 거치며 긴장감 있는 진행을 경험합니다. Gemini API와 Firebase를 활용해 실시간 텍스트 분석과 글로벌 랭킹 시스템까지 구현했습니다.

핵심 기능 & 로직

  • ✦

    Gemini API 실시간 텍스트 분석 엔진 — 감정이 배제된 논리적 구조일수록 높은 AI 유사도 점수 부여

  • ✦

    동적 채점 알고리즘 — 미션 정확도 비례 산출, 재시도 페널티 및 시간 경과 로그 함수 감점 로직 적용

  • ✦

    글로벌 실시간 랭킹 — Firestore 연동 실시간 동기화 시스템 구축

이슈

하드코딩된 로직으로 인한 신규 미션 확장 한계

해결 → 동적 맵핑 객체와 데이터 규격 표준화를 도입하여 유지보수성 극대화

🗺️ 한걸음

고도 데이터와 동적 기상 정보를 융합하여 거동 약자에게 계단과 급경사를 우회하는 안전 경로를 안내하는 AI 기반 배리어프리 복지 내비게이터

Python FastAPI OSMnx Leaflet.js OpenAI API
🐙 GitHub ▶ 시연영상

프로젝트 개요

거동이 불편한 어르신과 교통약자를 위해 계단·급경사를 우회하는 안전 경로를 안내하는 AI 기반 배리어프리 내비게이터입니다. 고도 데이터와 실시간 기상 정보를 결합해 경로의 위험도를 동적으로 계산하고, 챗봇과 음성 안내로 접근성을 높였습니다. FastAPI 백엔드와 Leaflet 기반 지도 시각화로 프론트-백엔드 연동을 리드했습니다.

핵심 기능 & 로직

  • ✦

    지형 데이터 매핑 — OSMnx 보행 네트워크와 OpenTopoData 표고 수치를 결합해 전 구간 경사도 산출 및 계단 위치 매핑

  • ✦

    가중치 기반 다익스트라 라우팅 엔진 — 강수량 등 기상 조건에 따라 계단·급경사(5~8% 초과) 구간 가중치 페널티를 동적으로 변형하는 안전 최우선 경로 탐색

  • ✦

    REST API JSON 명세 수립 — 우회된 계단 수·감소된 경사도 등 비교 수치 포함, 프론트-백엔드 연동 리드

  • ✦

    대화형 음성 가이드 — 행정 용어를 어르신 눈높이에 맞춘 OpenAI 챗봇 및 Web Speech API TTS 음성 안내

고도 데이터와 도로망 데이터 매핑 시 연산 병목 현상
